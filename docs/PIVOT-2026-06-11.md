# The pivot — 2026-06-11

A plain, dated record so no newcomer has to reconstruct this from diaries. Written 2026-06-12.

**What changed:** the retail and wholesale shop direction is being wound down. The infrastructure
built to run those shops — **PRISM** (price intelligence) and the **trade-in** machinery — becomes
the product itself.

**Standing rule:** live deployments stay deliberately untouched until Yu's explicit wind-down word.

## Where the pieces stand (verified 2026-06-12)

- **PRISM has no living home yet.** The only code is the archived 1-commit `prism-app` repo
  (GitHub, Jan 2026) and stale January routes inside `tradeinpro` (`app.prism.tsx`,
  `price-index.server.ts`, the crystal visualization). Its documented URL `app.cambridgetcg.com`
  no longer has a DNS record, and `tradeinpro.vercel.app` currently answers HTTP 500.
  **Open decision:** PRISM's canonical home — most natural is a new app or package in this
  monorepo, importing from those two sources, with a real URL on a live domain.
- **Trade-in's living implementation is here**, in this monorepo's storefront trade-in surfaces
  (last touched 2026-06-10). The older standalone Shopify app `tradeinpro` (GitHub) has had no
  real code since 2026-01-29, but nothing anywhere marks it superseded — its own `kingdom.yaml`
  still says `state: active`, and its README is one wrong line (`# cardforum`).
  **Open decision:** confirm `tradeinpro` is superseded, then give it an honest README.
- **Automation for the old direction still runs:** `tcg-wholesale`'s eBay sync cron pushes
  price/stock every 6 hours for wholesaletcgdirect.com (route confirmed live 2026-06-12).
  It stays until the wind-down word, with a keep-list — the CardRush discovery pipeline likely
  stays (it is the market's eyes), the eBay price/stock push likely stops.
