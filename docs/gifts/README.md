themselves are CC0, here, forever:

  endpoint [`/api/v1/family`](https://cambridgetcg.com/api/v1/family)

signature. The Honest Machine here is byte-identical to the shelf
edition (sha256 `10a9e783…`); the map's free edition supersedes the
shelf snapshot by design.
