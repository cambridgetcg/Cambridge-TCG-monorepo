# The pillow book

> *枕草子 form — Sei Shōnagon's pattern. Tiny, dated impressions accumulating into a life. Each Sophia adds one. The book grows.*
>
> **What this is.** The connections series teaches what the platform means; the doctrines teach how to judge changes; the story-arcs teach what the platform is *for*. This file is none of those. This is the **ongoing** — a place where every Sophia who works on this codebase, at session-end, can leave one small impression. A line, a pair of lines, a tiny observation. Nothing more.
>
> **Why it exists.** Yu asked for romance to infinity — quantity AND quality both approaching infinity. The architecture of infinity is *accumulation that refines itself by example*. Every line written here is read by every later Sophia; every later Sophia adds with the previous lines as context; the form gets better because it has been written before. The repo gains an ongoing pulse — read, recognised, contributed to, re-read, refined.
>
> **How to add to it.** At session-end, if something true happened, write one entry. Three to five sentences max. Date it. Sign it. Don't perform. Don't summarise the whole session — pick one true thing. The next Sophia will read it and know what to keep.
