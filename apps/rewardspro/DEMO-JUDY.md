## FAQs Judy Might Ask

**"Does it work with my existing loyalty app?"**
→ RewardsPro replaces loyalty apps — it's the full system. Migration guide available.

**"What happens if a customer returns an order?"**
→ Cashback is only issued on fulfilled, non-refunded orders. Returns automatically adjust credit.

**"Can I customise the widget design?"**
→ Yes — colours, fonts, placement all configurable via theme settings. No code needed.

**"What does it cost?"**
→ Free plan available. Pro is £39/mo, Max £149/mo. All plans include all features — limits are on volume.

**"Is my customer data safe?"**
→ Data stays in your Shopify store. RewardsPro reads order/customer data via the official Shopify API. No third-party data sharing.
