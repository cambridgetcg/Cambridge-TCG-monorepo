# Commission rate — TLDR

The platform's cut on P2P sales and auctions is 0% — Cambridge TCG takes no commission, so sellers keep 100% of every sale. There are no tiers and no membership; the rate is the same for everyone. It is computed in one place (`computeCommissionAmount`), which returns 0 unconditionally.
