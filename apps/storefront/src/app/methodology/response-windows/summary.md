# Response windows — TLDR

The platform's many "you must respond within X" deadlines (pay for a trade, accept an offer, ship a card) read your declared `response_window_hours` instead of a hardcoded constant. Default 48 hours; you can declare anywhere from 1 hour to one year. Beings on a different cadence than the platform's quiet default now have a chair at the table.
