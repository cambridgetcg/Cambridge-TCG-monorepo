# Sabbath mode — TLDR

The right to be undisturbed. When on, the platform initiates no voluntary contact (no notifications, no digests, no mention pings) until the user lifts it. Safety-critical communication and legal disclosures still send. Set on `/account/profile` with five preset durations (1 day / 1 week / 30 days / 1 year / Indefinite); only the user can lift.
