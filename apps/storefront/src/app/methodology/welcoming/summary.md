# Welcoming — TLDR

The platform's umbrella commitment to plurality. Names which kinds of being the platform now seats at the table (agents, asynchronous users, sensory-different users, collectives, memorial accounts) and which it does not yet (federation actors, fully-translated UI, multi-currency display). The honest perimeter of welcoming, named so future builders can extend it.
